JavaScript был создан для того чтобы сделать веб страницы "живыми"
Программы написанные на нём называются скриптами, могут встраиваться в HTML и выполняться автоматически при загрузке страницы

Им не нужна компиляция для запуска, это интерпретируемый язык

JS может выполняться везде где есть движок JS
У браузеров есть эти движки
V8 - Opera, Google
SpiderMonkey - Firefox 
Nitro, SquirrelFish - Safari
ChakraCore - Edge
Chakra, Triden - IE разных версий

Движок парсит скрипт
Компилирует его в машинный язык
После этого машинный язык запускается и работает довольно быстро

JavaScript безопасный язык, он не предоставляет низкоуровневый доступ к памяти или процессору

В браузере для JavaScript доступно все что связанно с манипулированием веб-страниц, взаимодействия с пользователем и веб-сервером

Например
- Изменять HTML разметку, модифицировать стили.
- Реагировать на действия пользователя, щелчи мыши, нажатие на клавиатуру
- Отправлять сетевые запросы, скачивать и загружать нужные файлы(AJAX и COMET)
- Получать и устанавливать куки, показывать сообщения, задавать вопросы посетителю сайта
- Запоминать данные на стороне клиента ("local storage")

JavaScript имеет ограничения, чтобы обезопасить посетителя от недобросовестного кода
- JavaScript не может читать\записывать файлы жесткого диска пользователя, не имеет прямого доступа к ОС
Есть ограниченный доступ к файлам, которые перетаскиваются в окно браузера либо выбираются с помощью input
Требуются разрешения на использование веб-камер и микрофона.
- "Политика одинакого источника". Окна не знают друг о друге и не могут между собой взаимодействовать


Плюсы:

- Полная интеграция с HTML/CSS
- Простые вещи делаются просто
- Поддерживаются всеми основными браузерами и включен по умолчанию

Есть языки "над" JavaScript, они быстро транспилируют свой синтаксис на JavaScript 
- CoffeeScript добавляет "синтаксический сахар" что позволяет писать более лаконичный код
- TypeScript добавляет строгую типизацию, для упрощения разработки и поддержки в больших и сложных систем
- Flow так же добавляет типизацию
- Dart имеет собственный движок, был предложен Google
